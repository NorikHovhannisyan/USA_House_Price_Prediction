# 🏡 USA House Price Prediction App

An interactive web application that predicts real estate prices in the United States using Machine Learning. This project demonstrates a full-stack Data Science workflow: from data cleaning and feature engineering to model deployment.

## 📊 Project Overview
The goal of this project is to provide accurate house price estimates based on key features like location, living space, and local socio-economic data. 

### Key Achievements:
* **MAE (Mean Absolute Error):** Successfully reduced from $153,000 to **$75,758** through iterative optimization.
* **Algorithm:** Implemented **XGBoost Regressor** for high-performance tabular data prediction.
* **Feature Engineering:** Created custom features like `Space_per_Room` to improve model sensitivity.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Libraries:** Scikit-learn, XGBoost, Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Deployment:** Streamlit
* **Version Control:** Git & GitHub

## 📂 Dataset Features
The model takes the following inputs to provide a prediction:
1. **Beds/Baths:** Number of bedrooms and bathrooms.
2. **Living Space:** Square footage of the home.
3. **Location:** Latitude and Longitude coordinates.
4. **Demographics:** Zip code population, density, and median household income.
